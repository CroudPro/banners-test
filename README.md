# AdriverBanners

## Installation
```bash
npm install
```

## Usage
```bash
npm run dev
```

## Build
```bash
npm run build
```

## Основные функции
- Показ баннеров
- Обработка серверных ошибок
- Отработка по событиям
- Поддержка мобильных устройств и iframe
- Поддержка легкого добавления новых событий
